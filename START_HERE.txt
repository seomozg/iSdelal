   → Восстановите данные Qdrant
   → Разработка локально 🚀

3️⃣  ВЫЛОЖИТЬ НА GITHUB:
   → Откройте: GITHUB_SETUP.md
   → Создайте репо
   → Пушьте код
   → GitHub Actions запустят тесты 🚀

4️⃣  ДЕПЛОИТЬ НА test-domain.ru:
   → Откройте: DEPLOY.md
   → Следуйте инструкциям
   → Настройте HTTPS
   → Live на test-domain.ru 🚀
